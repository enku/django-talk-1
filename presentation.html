What is a web framework?

  • Tool for serving HTTP requests
  • request → middleware → view → middleware → response
  • static media, authentication, database, templates, etc.

Django Views

  • Recieve a "request" and return a "response".
  • function- or class-based
  • Can serve web pages, REST APIs, media, etc.

Django ORM


from django.db import models


class Band(models.Model):
    """A model of a rock band."""
    name = models.CharField(max_length=200)
    can_rock = models.BooleanField(default=True)


class Member(models.Model):
    """A model of a rock band member."""
    name = models.CharField("Member's name", max_length=200)
    band = models.ForeignKey("Band")
                    

members = Member.objects.filter(band__name='Led Zeppelin')

for member in members:
    print(member.name)
                    

SELECT member.name FROM member 
INNER JOIN band ON member.band_id = band.id 
WHERE band.name = "Led Zeppelin";
